Django + Datastar

Hypermedia-driven web applications the simple way

Welcome! This is a collection of examples demonstrating how to build modern, reactive web applications using Django with Datastar.

Datastar brings the power of Server-Sent Events (SSE) and fragment-based updates to Django, letting you create dynamic UIs without writing JavaScript.
